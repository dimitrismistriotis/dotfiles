# dotfiles

## Checkout

Whenever needed assumes that it is cloned in the home directory.

```shell
cd ~
git clone git@github.com:dimitrismistriotis/dotfiles.git
```

Needed as some paths can only be hardcoded.

## Initialization

By running "initialize" script:

```bash
./initialize
```

then use `make`.
